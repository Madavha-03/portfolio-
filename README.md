Designing a multi-site network for 50+ users required me to plan IP addressing schemes, segment departments using VLANs, and configure dynamic routing with OSPF. I learned that every design decision has a real impact on network performance and security.

**Home Lab Network Segmentation & Security**
Building my own home lab showed me the importance of network isolation and security. Configuring pfSense to separate IoT, trusted, and guest traffic gave me practical firewall experience that goes beyond what textbooks teach. Setting up a VPN server further deepened my understanding of secure remote access.

**Network Traffic Analysis with Wireshark**
This lab changed how I see network communication. Capturing and filtering live packets helped me understand how protocols like HTTP, DNS, and ARP actually behave on a network. More importantly, it taught me how to identify suspicious traffic patterns — a critical skill for any network engineer or security professional.

**Crimping Project**
This hands-on task reminded me that networking is not only about software and configuration — physical infrastructure matters too. Properly terminating Cat6 cables and testing continuity taught me precision and attention to detail that is essential in a real server room environment.

**Active Directory Configuration**
Working with Active Directory gave me a deeper understanding of how user identity and access are managed in enterprise environments. Setting up domain services, creating organisational units, and applying Group Policy Objects showed me how IT administrators control security and access at scale. This experience connected my networking knowledge to systems administration, making me a more well-rounded IT student.

**SANReN Cybersecurity Challenge 2025**
Participating in a national competition was a turning point in my learning journey. Analysing PCAP files under time pressure forced me to apply multiple skills simultaneously — Wireshark dissection, protocol knowledge, and logical thinking. It showed me that I can perform in a competitive, real-world environment.
